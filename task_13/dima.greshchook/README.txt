...Уважаемый пользователь моего продукта, прочти несколько замечаний о данной версии.
1) Архитектура класса потобна Math из стандартного пакета java. Т.е. здесь много статических
методов, особенно это оправдано при использовании оппераций + - / * . Так как я нехотел 
менять значение операндов. Если нужно мы можем просто сослаться на результат который вернет
метод( Например Complex c = Complex.add(a, b); // еще лучше было бы в стиле С++
Complex c = a + b; )
2) Для нахождения корня 2-й степени предусмотрены два перегруженных метода:
	один для вещественных чисел,
	второй для комплексных
3) Нахождение корня производится для случая при k = 0. Хотя похорошему необходиму бы предоста-
вить интерфейс для нахождения n-го числа корней где k = 0, 1, 2, 3 ... n.
